# 💱 Currency Converter - Tkinter GUI App

A simple, clean, and functional **Currency Converter** built using Python's `tkinter` for GUI and the `CurrencyConverter` module for real-time currency conversion. Designed with simplicity in mind, this project allows users to convert between international currencies with just a few clicks!

---

## ✨ Features

- 📦 Converts currencies instantly using real exchange rates.
- 🖥️ User-friendly **GUI** with `tkinter`.
- 🔄 Converts from any currency (like `USD`, `INR`, `EUR`, etc.) to any other.
- 💡 Perfect for beginner developers to understand:
  - GUI programming
  - API-backed currency conversions
  - Event-driven logic

---

## 🎯 Use Cases

- Checking how much ₹1000 is worth in USD
- Understanding `tkinter` layout & widget placement
- Practicing Python GUI development
- Learning how to integrate external libraries with GUI

## 🚀 How It Works

1. **User enters an amount** (e.g., `100`).
2. **Select source currency** (e.g., `INR`).
3. **Select target currency** (e.g., `USD`).
4. Click the **Convert** button.
5. The app instantly displays the result at the bottom.
